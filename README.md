## Nuke OS is a simple commandline operating system built from concepts learnt from "Operating Systems : From 0 to 1" book

## Features implemented
1.Protected mode.
2.Creating and managing processes with TSS (Task State Structure).
3.Interrupts
4.LAPIC.
